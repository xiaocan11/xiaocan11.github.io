让我们来记住一下啊
hexo g (generate生成)
hexo s (server服务预览)
hexo d (deploy部署，好像我现在应该用不上了..)
hexo c (clean清理缓存，好像也是用不上)
cd “文件名称” (进入所输入的文件)
cd ../.. (降？文件级？，我懂就好)
hexo n “文章名称” (建文章，简单版)
hexo new post “文章名称” (建文章，充实版，为什么要用这个呢)
hexo n draft “草稿名称” (建草稿，建完要hexo s)
hexo s – draft (草稿显示于网站，不知道有没有记错)
hexo publish “草稿名字” (发布草稿，口令版，简单版是文件里复制黏贴)
OKie，暂时就这些，希望我能用的越来越娴熟^.^ ~
